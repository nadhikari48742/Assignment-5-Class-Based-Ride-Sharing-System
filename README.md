# Ride Sharing System

## Description
This project implements a Ride Sharing System using Pharo Smalltalk.

## Features
- Ride superclass
- StandardRide subclass
- PremiumRide subclass
- Driver class
- Rider class
- Fare calculation using polymorphism
- Ride assignment
- Ride request history
